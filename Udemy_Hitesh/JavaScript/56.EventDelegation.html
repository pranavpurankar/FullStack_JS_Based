Event Delegation in JavaScript

Event delegation is a technique in JavaScript where a single event listener is attached to a parent element to manage events on its child elements. Instead of adding individual listeners to each child, the event propogates (bubbles) up the DOM tree, and the parent handles it.

This approach is highly efficient, especially when dealing with dynamic elements added to the DOM after the page has loaded.


How it Works

  1. Event Bubbling:

    2. When an event occurs on element, it first triggers on the target element and then propogates upwards to its parent and ancestor elements.

  2. Using a Single Listener:

    3. By listening for an event on a common ancestor of multiple elements, you can determine the specific child element that triggered the event using the event.target property.



Advantages of Event Delegation

  1. Performance

    2. Reduces the number of event listeners, which saves memory and improves performance.

  2. Dynamic Elements

    3. Automatically works for elements added to the DOM after the listener is set.
