The fetch API is relatively new, it was introduced in the 2015. Before that everything was done using AJAX which stands for (Asynchronous JavaScript and XML). It is also important to understand the AJAX, although not widely used but should have strong understanding of it. MDN_Doc: XMLHttpRequest and MDN_ReadyState: Ready Status

Value State Description
0 UNSENT Client has been created. open() not called yet.
1 OPENED open() has been called.
2 HEADERS_RECEIVED send() has been called, and headers and status are available.
3 LOADING Downloading; responseText holds partial data.
4 DONE The operation is complete.
